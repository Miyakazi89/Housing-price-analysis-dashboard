# 🏠 Housing Price Analysis Dashboard

## 📊 Project Overview
This project focuses on data cleaning, exploratory data analysis (EDA), and visualization of housing price data. The goal is to identify key factors that influence property prices.

---

## 🎯 Objectives
- Clean and prepare raw datasets
- Perform univariate and bivariate analysis
- Identify relationships between features
- Build an interactive Power BI dashboard

---

## 🧰 Tools Used
- Python (Pandas, NumPy)
- Matplotlib & Seaborn
- Power BI

---

## 📂 Dataset
- Housing Dataset

---

## 🔍 Key Insights

### Housing Dataset
- Property size (area) strongly influences price
- More bedrooms and bathrooms increase property value
- Parking availability contributes to higher price
- Some outliers represent luxury properties

---

## 📌 Conclusion
The analysis highlights the importance of property size and key features in determining house prices. These insights can guide predictive modeling and real estate decisions.
